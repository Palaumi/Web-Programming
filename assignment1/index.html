My Projects

  • StoryPix
StoryPix is an android application where users can share photos and videos with their friends. In this app we have introduced timeline representation of photos and videos to avoid unnecessary scrolling when looking for old memories. Using this application a user can create a user profile, accept and send friend requests to another user. It also contains features like editing, search for friends and security to ensure the privacy of user. This app is designed using JSON parser, material design, Java, CSS and SQLite as its front end and back end tools.
